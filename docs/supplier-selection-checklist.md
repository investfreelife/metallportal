## Минимум приемлемого (must-have, все галочки)

- [ ] Прайс приходит регулярно, не реже 1×/неделю
- [ ] Прайс — табличный формат (Excel или CSV), не PDF и не картинка
- [ ] Колонки: артикул, наименование, цена, единица измерения — присутствуют
- [ ] Структура прайса не меняется (или меняется очень редко, и нас предупреждают)
- [ ] Хотя бы 30% наших SKU присутствуют в их прайсе
- [ ] Можем получить **3 последних прайса** для тестирования стабильности парсера

## Идеальный кандидат (nice-to-have)

- [ ] Прайс ежедневно автоматически (email или ЛК)
- [ ] Артикулы соответствуют ГОСТ
- [ ] Есть колонка остатка (in_stock количество)
- [ ] Готовы к API-интеграции в перспективе (6–12 мес)
- [ ] Имеют склад в МО или Москве (быстрая доставка)

- **Покрытие нашего каталога**: ____% SKU

## Что отдать на Week 2

После выбора положить в репо в `data/suppliers/<supplier-slug>/`:

```
data/suppliers/<slug>/
├── pricing-2026-04-XX.xlsx    # последний прайс
├── pricing-2026-04-YY.xlsx    # предыдущий — для теста парсера
├── pricing-2026-04-ZZ.xlsx    # ещё один — для проверки стабильности
├── sample.png                  # скриншот первой страницы прайса
└── notes.md                    # контакты, особенности формата
```

Содержимое `notes.md`:

```markdown
# Поставщик: <Название>

## Контакты
- Менеджер: ФИО, телефон, email
- ЛК: URL (если есть)
- Канал получения прайса: email / ссылка / FTP

## Формат прайса
- Файл: pricing-YYYY-MM-DD.xlsx
- Лист: Лист1 / Прайс / другое
- Заголовки в строке: 1 / 3 / другая
- Колонки (в порядке слева направо):
  - A — артикул
  - B — наименование
  - C — ГОСТ (опционально)
  - D — цена ₽/ед
  - E — единица измерения
  - F — остаток (опционально)
  - G — срок поставки

## Особенности
- Цена с НДС / без НДС
- Скидки от объёма (если есть — описать)
- Округление количества (например, кратно 6 м для арматуры)
- Минимальный заказ
- Город отгрузки

## Артикулы
- Совпадают с ГОСТ-обозначениями: да / нет / частично
- Свои внутренние коды: да / нет
- Связь с нашими SKU: точная / fuzzy / нет

## Стабильность формата
- Когда последний раз менялась структура: ____________
- Кто отвечает на вопросы по формату: ____________
```
