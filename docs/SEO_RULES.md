# Правила SEO для карточек товаров

## Шаблон title
{Название товара} купить в Москве — цена {цена} ₽/{единица} | МеталлПортал

## Шаблон description (155 символов)
Купить {название} в Москве по цене {цена} ₽/{единица}.
В наличии на складе. Доставка по РФ. Сертификаты. Оптом и в розницу.

## Шаблон SEO текста карточки (400-500 слов)
Структура:
1. H1: {Название} — цена, описание
2. Абзац 1: что это, ГОСТ, применение
3. H2: Характеристики
4. Таблица характеристик
5. H2: Как купить
6. Абзац: доставка, самовывоз, документы
7. H2: Почему МеталлПортал
8. Преимущества: склад в Москве, сертификаты

## Ключевые слова
- купить + название + в Москве
- название + цена
- название + оптом
- название + со склада
- название + ГОСТ

г. Москва, Походный проезд, 16
