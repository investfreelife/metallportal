# Забор из листа или из трубы: цена, монтаж, долговечность

Два массовых решения для частного периметра — забор из профнастила и забор из профильной трубы (3D-сетка, решётка, штакетник). Разница в стоимости погонного метра до 2-3 раз, но цена не всё: на ветреном участке профнастил парусит, на дачах с прозрачным проектом глухие ограждения запрещены без согласия соседей.

В гиде разбираем 4 типа конструкций, расчёт материалов на 50 м периметра (столбы / лаги / листы / сваи), палитру высот для разных целей (1.5 / 1.8 / 2.0 / 2.5 м) и правовое поле (СП 53.13330 для СНТ, муниципальные правила, ПДД для дороги).

Сводная таблица сценариев «когда профнастил, когда труба» и 5 FAQ закрывают типовые вопросы заказчика. Расчёт цены — в каталоге, по геометрии участка — у менеджера.
