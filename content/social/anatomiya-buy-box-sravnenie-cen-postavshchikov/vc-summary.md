# Buy Box B2B — продуктовый разбор алгоритма от Харлансталь

Buy Box — концепция Amazon (2007) для B2C, которая постепенно адаптируется в B2B-сегменте. В нашем B2B-маркетплейсе металлопроката Харлансталь видим как принципы B2C-Buy Box модифицируются под профессиональные закупки с большими чеками и регулярными отношениями с поставщиками.

В статье — технический разбор:

**7 факторов ранжирования** с типичными весами:
- TCO в точке выгрузки (40%)
- Срок поставки (15%)
- Рейтинг продавца от verified покупателей (15%)
- Полнота документов (паспорт + сертификат) (10%)
- Verified-статус (KYC платформы) (10%)
- Объём в наличии (5%)
- Дополнительные услуги (5%)

**TCO compare** — нормализация в итоговую цифру для покупателя. Поставщик с самой низкой ценой тонны может проиграть из-за дорогой логистики или плохого рейтинга.

**Spread цен** как индикатор рыночной ситуации:
- 5-15%: здоровая конкуренция
- 20-30%: разные уровни поставщиков
- >30%: red flag (вероятная ошибка в одном из предложений)

**Модификаторы алгоритма**: boost новым поставщикам (30 дней), penalty за устаревшие данные (>7 дней), throttling частых winner (для конкуренции).

**Эволюция B2C → B2B**: усложнились факторы (документы, отрасль), изменилась логика winner (TCO vs только цена), появилось эскроу для крупных контрактов.

В России B2B-Buy Box на металле начал формироваться в 2020-х. Доля сделок через маркетплейсы 15-25% → forecast 40-50% к 2028-2030.
