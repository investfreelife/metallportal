📐 Чертёж КМ vs КМД — снабженцу нужно знать оба

КМ = проект (тоннажи для тендера). КМД = детализация (точные размеры для заказа поставщику). Путаница = пересорт партии на сотни тыс. рублей.

3 ключевых факта:
• Маркировка: К-1 (колонна), Б-1 (балка), Ф-1 (ферма), С-1 (связь)
• Сварные швы по ГОСТ 2.312, болты по ГОСТ 2.315 — обязательно читать
• Болты для несущих узлов — минимум класс 8.8 (10.9 для преднапряжённых)
