# ГОСТ vs ТУ в металлопрокате: техническая разница и выбор для инженерного проекта

При закупке металлопроката для технических проектов снабженец регулярно видит две альтернативы: позицию по ГОСТу и аналог «по ТУ» с разницей в цене 8-20%. На поверхности они выглядят идентичными, но юридически и технически это разные классы документов. В одних проектах ТУ-прокат оптимален, в других ведёт к отказу Госстройнадзора. В этой статье — техническая разница и алгоритм выбора для инженерного проекта.

## Принципиальная разница

**ГОСТ (государственный стандарт)** — нормативный документ, разработанный коллективно отраслью через профильный комитет Росстандарта. Действует на всей территории РФ. Имеет два класса:
- **Обязательный** — для продукции с прямой санкцией за нарушение (продукция оборонки, медицины, отдельных категорий пищёвки).
- **Добровольный** — применяется по согласию сторон. Если в договоре стоит ссылка на конкретный ГОСТ — он становится обязательным для этой сделки.

**ТУ (технические условия)** — нормативный документ, разработанный **самим заводом-производителем** под свою продукцию. Утверждается руководителем предприятия, регистрируется в Росстандарте без независимой экспертизы. Действует только для этого завода.

## Различия в допусках

Реальные технические расхождения по параметрам:

| Параметр | ГОСТ (типовой) | ТУ (типовой) | Импакт |
|---|---|---|---|
| Допуск по толщине (лист 4 мм) | ±0.4 мм (10%) | ±0.6 мм (15%) | Геометрия |
| Содержание серы (Ст3) | ≤ 0.05% | до 0.06% | Свариваемость |
| Содержание фосфора | ≤ 0.04% | до 0.05% | Ударная вязкость |
| Предел текучести (мин) | гарантированный | средне-плавочный | Расчёт нагрузок |
| Cэкв для свариваемости | контролируется | не всегда контролируется | Сварные соединения |

## Где обязателен ГОСТ — без вариантов

Категории, где ТУ-аналог не пройдёт авторский надзор:

1. **Несущие конструкции зданий** — СП 16.13330 (стальные), СП 63.13330 (железобетонные)
2. **Мостовые сооружения** — СП 35.13330
3. **Сосуды и аппараты под давлением** — ГОСТ 34347-2017, контроль Ростехнадзора
4. **Объекты транспортной инфраструктуры** — ПДД + специальные ГОСТы
5. **Военные объекты** — обязательный пакет ГОСТ + ОСТ + военная приёмка
6. **Ядерная энергетика** — ГОСТ + ПНАЭ + лицензирование
7. **Объекты с госзаказом** — типовые контракты с прямой ссылкой на ГОСТ
8. **Экспорт в ЕАЭС** — таможня требует ГОСТ-маркировку

## Где ТУ оправдано

- Хозяйственные постройки (сараи, навесы, теплицы)
- Заборы для дач и частного сектора
- Внутренние конструкции без расчётной нагрузки (стеллажи, перегородки)
- Декоративные элементы (поручни, ворота)
- Временные сооружения (срок службы до 3 лет)
- Опытные и единичные изделия
- Ремонт существующих ТУ-конструкций

## Алгоритм для проектировщика

```
1. Объект имеет проектную документацию со ссылкой на ГОСТ?
   ├─ ДА → только ГОСТ-партия. Замена = лист авторского надзора.
   └─ НЕТ → переход к шагу 2.

2. Объект попадает под обязательную сертификацию по ТР ТС?
   ├─ ДА → проверить применимость ТУ через орган сертификации.
   └─ НЕТ → переход к шагу 3.

3. Какая ожидаемая нагрузка / срок службы?
   ├─ Несущая / >10 лет → ГОСТ предпочтительней (документально проще).
   └─ Не несущая / краткосрочно → ТУ возможен, проверить допуски.
```

## Контрактная дисциплина

В договоре поставки явно прописать одну из формулировок:

- **«Только ГОСТ XXXX-YYYY»** — для критичных проектов, замена недопустима.
- **«ГОСТ XXXX-YYYY или эквивалент по EN/ASTM»** — допускает импортные аналоги.
- **«ГОСТ XXXX-YYYY или ТУ конкретного производителя с допусками не хуже ГОСТовских»** — компромисс с явным контролем.

Без формулировки = поставщик может прислать любой ТУ-аналог формально соответствующий маркировке.
