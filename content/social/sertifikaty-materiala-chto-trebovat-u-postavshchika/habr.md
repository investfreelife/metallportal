# Сертификация металлопроката — compliance чеклист для B2B-инженера

Сертификация металлопроката в России регулируется системой документов разного юридического веса и сферы применения. Без правильного пакета претензии по качеству не работают в суде, Госстройнадзор отказывает в приёмке, экспорт встаёт на таможне. В этой статье — техническая систематика 9 видов сертификатов и алгоритм выбора пакета под проект.

## Иерархия сертификатов — юридический вес

```
ТР ТС / ТР ЕАЭС (обязательный)
   ↓
Сертификат соответствия (от аккредитованного органа)
   ↓
Декларация о соответствии (от заявителя)
   ↓
Паспорт качества (от завода на партию)
```

## 9 типов сертификатов

**Базовые российские:**

1. **Паспорт качества** — заводской документ на конкретную плавку. Содержит химсостав, мехиспытания, номер плавки, подпись ОТК. **Самый детальный**, но привязан к партии.

2. **Сертификат соответствия** — от аккредитованного органа сертификации после испытаний образцов. Выдаётся на товарную группу. Действует 1-3 года. Проверка подлинности через **fsa.gov.ru**.

3. **Декларация о соответствии** — упрощённая форма, оформляется заявителем. Регистрируется ОС без независимой экспертизы. Для большинства массового металлопроката достаточно.

**Технические регламенты:**

4. **ТР ТС / ТР ЕАЭС** — обязательная сертификация для определённых групп продукции:
   - ТР ТС 010/2011 — машины и оборудование (готовые металлоконструкции: краны, ёмкости)
   - ТР ТС 030/2012 — смазочные материалы (для металлообработки)
   - ТР ЕАЭС 049/2020 — взрывозащищённое оборудование

Прокат (лист, труба, арматура) обычно НЕ подпадает под обязательный ТР ТС. Но готовые изделия из этого проката — часто да.

**Сертификаты происхождения:**

5. **СТ-1** — для торговли в ЕАЭС (Россия, Беларусь, Казахстан, Армения, Киргизия). Выдаётся ТПП. Без СТ-1 = полные пошлины как для третьих стран.

6. **EUR.1** — для экспорта/импорта со странами преференциальной торговли (Сербия, Иран, Вьетнам).

7. **Form A** — для экспорта в страны GSP. Применение ограничено санкционными режимами.

**Отраслевые:**

8. **ASME / Lloyd's Register** — оборудование под давлением, морские конструкции (международный стандарт).

9. **HACCP** — пищевая безопасность (металл в контакте с пищей).

## Алгоритм выбора пакета по проекту

```
Внутренний рынок РФ, стандартная стройка:
  ✅ Паспорт качества + Сертификат соответствия / Декларация

Объект под Госстройнадзор:
  ✅ Базовые + СРО проектной организации + лицензия на сосуды (если применимо)

Импорт из ЕАЭС:
  ✅ Базовые + СТ-1 + таможенная декларация

Экспорт из РФ:
  ✅ Базовые + EUR.1 / Form A + контракт с инкотермсами

Оборонный заказ:
  ✅ Базовые + военная приёмка (5-я форма) + ФСБ/ФСТЭК

Фарма / биотех:
  ✅ Базовые + ASTM A240 / EN 10088 на нержавейку с электрополировкой
```

## Цифровизация сертификации

С 2020-х:
- **Единый реестр Росаккредитации** (fsa.gov.ru) — публичная база, поиск по номеру 30 сек.
- **QR-коды** на бумажных сертификатах → моментальная проверка через смартфон.
- **Электронная подпись органа сертификации** (КЭП), проверка через КриптоПро.
- **API-интеграция с ERP** — автоматическая верификация каждой партии.

Подделка отлавливается за 1 минуту. По 327 УК РФ + аннулирование Росаккредитацией.
