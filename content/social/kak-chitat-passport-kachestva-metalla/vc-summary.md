# Как мы научились ловить подделки паспортов качества за 5 минут

В B2B-маркетплейсе металлопроката Харлансталь регулярно сталкиваемся с тем, что серые поставщики используют поддельные паспорта качества — крадут реквизиты реальных заводов, копируют типовые данные химсостава. Без навыка чтения этого документа покупатель не имеет защиты при браке (паспорт = основа всех претензий).

В статье — операционный навык для снабженца, который занимает 10-15 минут на партию и закрывает 95% рисков:

1. **Структура настоящего паспорта** — обязательные поля (шапка / идентификация плавки / химсостав / мехиспытания / подписи).

2. **Чтение химического состава** — формула углеродного эквивалента (Cэкв), интерпретация серы/фосфора как «вредных» примесей, проверка через предельные значения ГОСТ.

3. **Механические характеристики** — предел текучести vs временное сопротивление (соотношение характеризует пластичность), относительное удлинение, ударная вязкость при отрицательных температурах.

4. **5 красных флагов подделки** — круглые цифры химии (0.20% / 0.50%), отсутствие живой подписи ОТК, все характеристики на границе ГОСТ-минимума, артефакты PDF-склейки.

5. **Алгоритм проверки** — прямой звонок в ОТК завода по номеру плавки даёт подтверждение или опровержение за 5 минут.

Подделка отлавливается с минимальными издержками. По 327 УК РФ — до 2 лет лишения свободы — серьёзный леверидж для возврата денег.
