# Аналоги марок стали: техническое соответствие ГОСТ ↔ EN ↔ ASTM ↔ JIS

При работе с импортным металлом, чтении зарубежных чертежей и подборе российских заменителей инженер регулярно сталкивается с задачей перевода марок между национальными стандартами. На первый взгляд таблицы соответствия выглядят прямолинейно — но дьявол в деталях: расхождения по химии, допускам, классам качества могут сделать формально-аналог технически неподходящим.

## Системы маркировки — формальная логика

**ГОСТ (Россия):** буквенно-цифровая система с расшифровкой химии в названии.
- `09Г2С` = 0.09% C, ~2% Mn (Г), Si (С)
- `08Х18Н10` = 0.08% C, 18% Cr, 10% Ni
- `40Х` = 0.40% C, легированная Cr

**EN (Европа):** «S» + предел текучести в МПа + категория качества.
- `S235JR` = Steel, σт=235 МПа, JR = ударная вязкость 27 Дж при +20°C
- `S355J2` = S355 + KCU при -20°C
- Нержавейка: числовое обозначение **1.4301** (= AISI 304), **1.4404** (= 316L)

**ASTM (США):** «A» + номер стандарта + Grade.
- `ASTM A36` — конструкционная углеродистая
- `ASTM A572 Grade 50` — высокопрочная низколегированная (50 ksi = 345 МПа)
- Нержавейка — серия AISI: 304, 316, 430

**DIN (старый ФРГ):** заменён EN, но в обиходе. `St37-2`, `X5CrNi18-10`.

**JIS (Япония):** `SS400`, `SUS304`, `SCM440`.

## Ключевые пары для 80% задач

```
Углеродистая массовая:
Ст3сп ↔ S235JR ↔ ASTM A36 ↔ SS400

Низколегированная конструкционная:
09Г2С ↔ S355J2 ↔ A572 Gr 50 ↔ SM490

Нержавейка пищёвая:
AISI 304 ↔ 1.4301 ↔ 08Х18Н10 ↔ SUS304

Нержавейка морская / фарма:
AISI 316L ↔ 1.4404 ↔ 03Х17Н14М3 ↔ SUS316L
```

## Подводные камни взаимозамены — engineering perspective

**1. Несоответствие по узким параметрам химии.**
09Г2С и S355J2 идентичны по C/Mn/Si, но S355J2 имеет жёсткое требование по N (≤0.012%). 09Г2С — нет. Для сосудов под давлением это критично.

**2. Допуски геометрии.**
EN 10025 на лист: допуск по толщине ±0.5-0.8 мм. ГОСТ 19903-2015: ±0.4-0.6 мм. Партия по марке аналог, по геометрии может не пройти ОТК ГОСТ-проекта.

**3. Свариваемость и Cэкв.**
Углеродный эквивалент: `Cэкв = C + Mn/6 + (Cr+Mo+V)/5 + (Ni+Cu)/15`

Для свариваемых конструкций Cэкв ≤ 0.50 — без подогрева. ≥ 0.55 — с предварительным подогревом + специальными электродами. Аналоги должны совпадать по Cэкв, не только по марке.

**4. Старые vs новые редакции стандартов.**
ГОСТ 380-71 и ГОСТ 380-2005 — одна Ст3сп с разными требованиями. EN 10025-1 vs 10025-2 — разные пакеты испытаний. При переписке стандарта на проекте — проверять конкретную редакцию.

**5. Контрактная дисциплина.**
В договоре «Сталь Ст3сп по ГОСТ 380» + поставка с маркировкой S235JR = формально нарушение, даже если химия совпадает. Допсоглашение или явная формулировка «или эквивалент по EN/ASTM».

## Алгоритм для проектировщика при подборе аналога

```
1. Сверка химии по паспорту качества партии (не «по таблице»)
2. Проверка Cэкв если конструкция сварная
3. Проверка допусков геометрии под требования проекта
4. Согласование с проектировщиком в письменной форме
5. Документирование в исполнительной документации
```
