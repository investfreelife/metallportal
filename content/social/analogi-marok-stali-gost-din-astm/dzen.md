# Аналоги марок стали: таблицы соответствия ГОСТ vs DIN vs ASTM vs EN

При работе с импортным металлом, чтении зарубежных чертежей и подборе российских заменителей снабженец каждый раз сталкивается с задачей: «какая русская марка соответствует европейской S235JR или американской ASTM A36?». В этом гиде даём проверенные таблицы аналогов по основным группам сталей.

## Системы маркировки

- **ГОСТ (Россия/СНГ)** — формула с расшифровкой химии: «09Г2С» = 0.09%C, 2%Mn, Si.
- **EN (Европа)** — формат «S» + предел текучести: «S355J2» = σт 355 МПа, KCU при -20°C.
- **ASTM (США)** — номер стандарта + Grade: «ASTM A572 Grade 50» = σт 50 ksi (345 МПа).
- **DIN (старый ФРГ)** — заменён EN, но в обиходе: «St37-2».
- **JIS (Япония)** — «SS400» / «SUS304».

## Ключевые пары на 80% задач

**Углеродистая массовая:**
- Ст3сп ↔ S235JR ↔ ASTM A36 ↔ SS400
- Для несущих болтированных и сварных конструкций общего назначения

**Низколегированная конструкционная:**
- 09Г2С ↔ S355J2 ↔ A572 Gr 50 ↔ SM490
- Мосты, тяжёлые металлоконструкции, сосуды

**Нержавейка пищёвая:**
- AISI 304 ↔ 1.4301 ↔ 08Х18Н10 ↔ SUS304
- Молочное оборудование, фасады, поручни

**Нержавейка морская/фарма:**
- AISI 316L ↔ 1.4404 ↔ 03Х17Н14М3 ↔ SUS316L
- Реакторы, рассолы, морские конструкции

## Подводные камни взаимозамены

- Аналог = первое приближение, **не финальное решение**
- Перед заменой — проверка химии по паспорту качества партии
- Согласование с проектировщиком письменно
- Контрактная дисциплина: «ГОСТ» в договоре + поставка «по EN» = нарушение
