## Body content (SEO 200-250 слов)

H2: **Какие виды двутавров есть в России**

- ГОСТ 8239-89 — обычные с уклонными гранями полок (старый стандарт, ещё в обиходе)
- ГОСТ 26020-83 — с параллельными гранями (Б балочные, Ш широкополочные, К колонные)
- ГОСТ Р 57837-2017 — современный стандарт, заменяющий 26020 (расширенная номенклатура)

H2: **Различия Б / Ш / К**

Б (балочный) — для пролётных балок, относительно высокий узкий. Ш (широкополочный) — для перекрытий, h≈b. К (колонный) — для вертикальных стоек, толстые полки и стенка.

H2: **Расчёт балки для перекрытия**

Краткие примеры: для пролёта 6 м и нагрузки 400 кг/м² типично 25Б1 / 30Б1; для пролёта 9 м — 35Б2 / 40Б1.
