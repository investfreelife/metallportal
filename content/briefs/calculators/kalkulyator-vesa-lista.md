# Content Brief: Калькулятор веса листа

## Цель страницы

SEO landing + рабочий инструмент. Вводит толщину + длину × ширину + количество листов → получает массу партии. Покрывает запросы «калькулятор веса листа», «масса листа стального», «сколько весит лист 3 мм».

## UI / форма

Поля:
- **Материал** — select: Сталь углеродистая (7850 кг/м³), Нержавейка AISI 304/316 (7950), Алюминий (2700), Медь (8920), Латунь (8500), Оцинковка (7850)
- **Толщина (мм)** — number, default 3
- **Длина (мм)** — number, default 2500
- **Ширина (мм)** — number, default 1250
- **Количество листов (шт)** — number, default 10
- Кнопка / live-расчёт

Output:
- Вес одного листа (кг)
- Вес партии (кг + тонн)
- Площадь партии (м²) — для лазерной резки расчётов
- Объём перевозки (м³) — для логистики

## Формула

```
Вес 1 листа (кг) = (длина_м × ширина_м × толщина_м) × плотность_кг_м³
```

Плотности из выпадающего «материал».

Стандартные размеры листов для предзаполнения:
- Сталь: 1250×2500, 1500×3000, 1500×6000
- Нержа: 1000×2000, 1250×2500, 1500×3000
- Алюминий: 1000×2000, 1250×2500
- Оцинковка: 1000×2000, 1250×2500

## Body content (SEO, 200-300 слов)

H2: **Как пользоваться калькулятором веса листа**

Калькулятор работает по простой формуле V × ρ = m. Введите габариты листа и материал — получите вес. Поддерживает популярные материалы листового проката: углеродистую сталь, нержавейку, алюминий, медь, латунь, оцинковку.

H2: **Зачем считать массу листа**

Снабженцу — для сверки накладной поставщика (заявленная масса vs расчётная). Прорабу — для подбора подъёмных механизмов и автотранспорта при доставке. Для лазерной резки — расчёт стоимости по площади листа.

H2: **Стандартные размеры листового проката**

Краткая таблица ходовых форматов по материалам (см. UI выше) — для long-tail запросов типа «вес листа 1500×6000×4 мм».
