## UI / форма

Поля:
- **Диаметр (Ø, мм)** — select с опциями из ГОСТовского ряда: 6, 8, 10, 12, 14, 16, 18, 20, 22, 25, 28, 32, 36, 40
- **Длина одного стержня (м)** — number input, default 11.7 (мерная), допустим decimal
- **Количество стержней (шт)** — number input, default 100
- Кнопка **«Рассчитать»** (или live-расчёт on change)

Output:
- **Вес одного стержня (кг)** — заметно крупным шрифтом
- **Вес партии (кг)** — общий
- **Вес партии (тонн)** — для оптовых заказов
- **Погонный метраж (общий)** — длина × количество (полезно для нарезки)

Бонус-кнопка: **«Добавить позицию»** → можно посчитать смесь нескольких диаметров одним заказом → итоговый вес.

## Формула

```
Погонный вес арматуры (кг/м) = π × (диаметр/2000)² × 7850 кг/м³
где 7850 — плотность стали.
```

## Body content (200-300 слов под формой для SEO)

H2: **Как пользоваться калькулятором веса арматуры**

Калькулятор работает по табличным данным ГОСТ Р 52544-2006. Введите диаметр (от Ø6 до Ø40 мм), длину одного стержня (стандарт — мерная 11.7 м или 12 м) и количество стержней — получите вес одного стержня и общий вес партии в кг / тоннах.

H2: **Зачем считать вес арматуры**

Снабженцу — для сверки накладной поставщика (заявленная масса vs расчётная по фактическому количеству). Расхождение 2-5% — норма (рифление, окалина). Расхождение >5% — основание для претензии о недовесе.

Прорабу — для расчёта тоннажа на проект, выбора грузоподъёмности автотранспорта, обоснования заказа.

Бухгалтерии — для оприходования партии по плановым весам и проверки счёта-фактуры.

H2: **Погонный вес по диаметру — таблица**

Таблица повторяет ту что в форме, для индексации поисковиками. Каждая строка — отдельный параграф для long-tail запросов «вес арматуры 12 мм за метр», «сколько весит арматура 16 мм 11.7 м».
