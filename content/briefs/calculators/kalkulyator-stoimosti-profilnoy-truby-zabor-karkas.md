## UI / форма — 2 wizard's с tabs

### Tab «Забор»:

- **Длина периметра (м)** — number, default 50
- **Высота забора (м)** — select: 1.5, 1.8, 2.0, 2.5
- **Шаг столбов (м)** — select: 2.0, 2.5, 3.0 (default 2.5)
- **Тип столбов** — select:
  - Профтруба 60×60×2 (стандарт)
  - Профтруба 60×60×3 (усиленный)
  - Профтруба 80×80×3 (тяжёлый, ветреный участок)
- **Заполнение** — select:
  - Профлист (профтруба только каркас 40×20)
  - Сварная 3D-сетка (нет лаг, только столбы + хомуты)
  - Решётка из проф трубы 20×20 (заполнение из трубы)

Output:
- Количество столбов (шт) + общий метраж (пог.м)
- Количество лаг (если применимо) + общий метраж
- Количество хомутов / метизов (шт)
- Общий вес партии (тонн)
- Pricing range: ×1 (base economy с профлистом) vs ×1.8-2.5 (3D-сетка) vs ×2-3 (решётка с покраской)

### Tab «Каркас дома»:

- **Габариты по осям (м×м)** — preset: 6×8, 8×10, 10×12, 10×15, custom
- **Этажность** — select: 1, 1 + мансарда, 2
- **Шаг стоек (м)** — select: 1.5, 2.0, 3.0 (default 2.0)
- **Тип сечения стоек** — select: 80×80×3, 100×100×4, 120×120×4
- **Тип обвязки** — select: швеллер №16 / двутавр 14Б1 / профтруба 100×50×4

Output:
- Вертикальные стойки (шт + метраж)
- Горизонтальные ригели (метраж)
- Обвязка (метраж)
- Подкосы / раскосы (метраж)
- Анкерные болты (шт + спецификация)
- Общий вес партии (тонн)
- Pricing range relative

## Формула / логика расчёта

Для **забора** (Тab «Забор»):
```
Кол-во столбов = round(длина / шаг_столбов) + 1 (угловой)
Метраж столбов = кол-во × (высота + 1 м заглубления)
Метраж лаг = длина × 2 (2 ряда лаг)
Кол-во хомутов = (длина / 2.5) × 4 (4 на секцию)
```

Для **каркаса** (Тab «Каркас»):
```
Кол-во стоек по периметру = (2×(длина+ширина) / шаг_стоек) + 4 угловых
Кол-во внутренних стоек = (площадь / 25) — для несущих стен (правило 1 стойка на 25 м²)
Общий метраж стоек = кол-во × высота_этажа × этажность
Метраж обвязки (низ + верх + межэтажная) = периметр × (этажность + 1)
Метраж ригелей + подкосы = 30-40% от метража стоек
```

Pricing range relative based на типовом profile selected.

## Body content (SEO 250-300 слов)

H2: **Сколько профильной трубы нужно на забор 50 м**

Готовый ответ для самой массовой ситуации (50 м, высота 2 м, шаг 2.5 м):
- 21 столб 60×60×2 длиной 3 м = 63 пог.м
- 100 пог.м лаги 40×20×2 (2 ряда)
- 350 саморезов + 21 заглушка

Для типового профлиста = ~250 кг металла на 50 м забор. Подробности — в [гиде по выбору забора](/blog/list-ili-truba-dlya-zabora-sravnenie).

H2: **Сколько трубы на каркас дома 8×10 м**

Готовый ответ для типового одноэтажного:
- ~30 стоек 100×100×4 высотой 3 м = ~90 пог.м
- Обвязка периметром = ~72 пог.м швеллера №16
- Ригели = ~50 пог.м

Итого ~1.5-2 тонны металла на каркас одноэтажного дома 8×10. Двухэтажный = 3-4 тонны.

H2: **Чем wizard отличается от ручного расчёта**

Снабженец вводит 5 ключевых параметров вместо 30 позиций спецификации — алгоритм сам собирает готовый комплект по типовому шаблону. Экономит 30-60 минут на проект.
