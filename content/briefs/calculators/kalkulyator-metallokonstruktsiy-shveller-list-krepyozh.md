# Content Brief: Калькулятор металлоконструкций (швеллер + лист + крепёж)

## Цель страницы

SEO landing для **multi-item калькулятора** — пользователь добавляет несколько позиций (швеллер + лист + метизы) в один расчёт, получает общую массу, спецификацию для накладной и pricing range. Замена excel-таблиц для типового сметчика / снабженца.

**Wordstat:** «калькулятор металлоконструкций» ≈ 9K/мес + «спецификация металлоконструкций онлайн» ≈ 5K = ~14K combined.

## Отличие от других calc briefs

Briefs #1-7 — single-item calculators (один тип проката). Этот **multi-item BOM (Bill of Materials)** — пользователь собирает спецификацию из 5-15 позиций как в excel, получает итог. Замена ручного составления спецификации.

## UI / форма — multi-item table

**Структура:** редактируемая таблица «Позиции» с кнопкой «+ Добавить позицию».

Каждая строка таблицы:
- **Тип проката** — select: Швеллер / Двутавр / Уголок равнополочный / Уголок неравнополочный / Лист / Профильная труба / Круглая труба / Арматура / Круг / Полоса / Метизы (болт+гайка+шайба) / Сварочные материалы
- **Сечение / диаметр / толщина** — input зависит от типа (select для standardize ГОСТ, custom для нестандарта)
- **Длина / габариты** — input
- **Количество** — number
- **Кол.во единиц** — auto (кг / шт / м²)
- **Расчётный вес 1 ед.** — auto (по формулам / ГОСТ табл.)
- **Общий вес позиции** — auto
- **🗑** — удалить строку
- **📋** — дублировать (для повторов сечения с разной длиной)

Под таблицей:
- **+ Добавить позицию** — кнопка
- **+ Загрузить из шаблона** — preset комплекты (типовой каркас одноэтажки / типовой забор 50м / типовой навес 6×4)
- **+ Импорт CSV/Excel** — для существующих спецификаций (опционально)

Output блок:
- **Итого по типу проката** (швеллер X тонн + лист Y тонн + метизы Z кг)
- **Общая масса партии** (тонн)
- **Pricing range** относительно (low/median/high estimate)
- **Spec PDF** — кнопка экспорта спецификации для отправки поставщику

## Backend logic

Каждый тип проката использует свою формулу / табл. ГОСТ (из briefs #1-7 уже описаны):

| Тип | Источник весов |
|---|---|
| Арматура | brief #1 (ГОСТ Р 52544) |
| Лист (углерод) | brief #2 (формула V×ρ) |
| Лист (нержа) | brief #6 (плотности 8 марок) |
| Труба круглая/профильная | brief #3 (формулы + ГОСТ 30245) |
| Уголок | brief #4 (ГОСТ 8509/8510) |
| Двутавр / Швеллер | brief #5 (ГОСТ 8239/26020/8240) |
| Метизы | hardcode табл. (ГОСТ 7798 болты + 5915 гайки) |
| Сварочные материалы | nominal вес электродов / проволоки |

Pricing range = база цен на каждый тип в каталоге × количество.

## Body content (SEO 250-300 слов)

H2: **Что такое калькулятор металлоконструкций**

Multi-item инструмент для сметчика / снабженца, который собирает спецификацию проекта из 5-15 позиций и получает итоговую массу + стоимость без ручного пересчёта в excel. Заменяет 30-60 минут работы 5 минутами интерфейса.

H2: **Типовые сценарии использования**

- **Снабжение по чертежу КМД** — переводим спецификацию КМД в готовый заказ поставщику с проверенными весами.
- **Сметная подготовка** — оценка стоимости металлоконструкции для тендера без затрат времени на полный расчёт.
- **Замена позиции на аналог** — пользователь видит как изменение марки или сечения влияет на итоговую массу и стоимость.

H2: **Чем wizard отличается от excel**

Excel требует: ручной ввод формул, хранение справочников плотностей/ГОСТ-таблиц, постоянное обновление цен. Wizard: всё встроено, обновляется автоматически, валидация ошибок (например, нестандартный размер арматуры подсвечивается).

H2: **Шаблоны типовых конструкций**

5 готовых пресетов:
- Забор 50 м (2 м высота, проф труба + профлист)
- Каркас одноэтажки 8×10 (швеллер + лист обвязки + метизы)
- Навес 6×4 (двутавр + профлист + хомуты)
- Хозблок 3×4 (уголок + лист + крепёж)
- Лестница металлическая (швеллер косоуры + лист ступени + перила)

Пользователь выбирает preset → автоматически заполняется 8-15 позиций → можно поправить и сохранить.
