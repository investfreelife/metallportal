# External Telegram-каналы для cross-posting

> **Goal:** найти 8-12 каналов где можно бесплатно (без оплаты VC) опубликовать наш контент или анонс с ссылкой. Сегментация: тематические B2B-каналы (стройка, металл, снабжение) + общие бизнес-каналы.
>
> **Disclaimer:** список собран по открытым источникам (Yandex / TGStat-like ресурсы / отраслевые сообщества). Размер аудитории и условия posting'а могут меняться. Перед каждой подачей — проверка актуальности через @SearchExpressBot, TGStat.ru, Telemetr.io.

## Posting strategy

**Week 1 — Day 4-7 (после Day 1-3 на собственных каналах):**
- Отправить submission в 3-4 топовых канала (1, 4, 6, 9) с правильным форматом подачи (предложение гостевого поста / cross-post с упоминанием).
- Ждать ответы 1-2 недели (нормальный срок для admin'ов).

**Week 2-3:**
- Расширить на 4-5 каналов сегментов 2-3.
- Tracking views/leads через UTM-метки в URL (`?utm_source=tg&utm_campaign=external&utm_content=channel_name`).

**Week 4+:**
- На основе реакции (отвечают / не отвечают / какие темы лучше идут) — focus на топ-3 каналах с regular cross-posting.

Template сообщения в DM админу:

```
У нас 16 экспертных гайдов для снабженцев и строителей

Предлагаю гостевой пост для вашей аудитории на тему 
«[тема]» — без рекламы, с фокусом на экспертный контент, 
с подписью «материал подготовлен Харлансталь» + ссылкой.

Готов прислать draft на проверку, или вы можете 
взять готовый материал с нашего блога (с указанием 
авторства).

Спасибо!
```

## Метрики для tracking

После публикации в external канале — записывать в `publication-log.csv` (с пометкой `channel: tg_external_<name>`) + tracking через UTM:

- Views (TGStat.ru или ручной)
- Forwards (показатель engagement)
- Click-through к harlansteel.ru (UTM в аналитике сайта)
- Leads attributed (если приходят через форму)
