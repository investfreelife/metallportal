# Distribution Playbook — Harlanmetall content

> **Этот документ — turnkey пошаговый гид для Sergey'я по размещению 16 articles на 6 бесплатных площадках.**
>
> **Принцип:** Юля готовит turnkey content + пошаговые инструкции, Sergey **только клик-paste**. Ничего разбираться не надо.
>
> **Время для Sergey:** Day 1 = 35 минут (регистрация 4 каналов + 4 первых публикации). Далее = 5-10 минут на каждый последующий post.

---

## Quick start — Day 1 (35 минут total)

**Перед началом** — открой `content/distribution/ready-to-publish/` и убедись что там лежат 10 готовых файлов с расширением `.md`. Каждый файл = готовый post для одного канала. Если папка пустая — это значит Юля ещё не закончила Phase 3, пинг ей.

**Последовательность Day 1:**

| Шаг | Канал | Действия | Время |

## 1. Я.Дзен — 5 минут регистрация + 3 минуты публикация

### Регистрация канала

1. Открой https://dzen.ru/ в браузере (где залогинен в Yandex ID, обычно тот же что Yandex.Почта).
2. Если ещё не залогинен — войди под существующим Yandex ID (рекомендую `7909885@mail.ru` если он привязан к Yandex; иначе любой твой Yandex логин).
3. В правом верхнем углу — иконка профиля → выбор «Создать канал».
4. Тип канала: выбери **«Канал автора»** (не «Сообщество» — оно для модерации).
5. Заполни форму:
   - **Название:** `Харлансталь — гид по металлопрокату`
   - **Описание:** `B2B-маркетплейс металлопроката. Гайды для снабженцев, строителей, проектировщиков. ГОСТы, расчёты, безопасная закупка.`
   - **Аватар:** загрузи logo Харлансталь (если нет под рукой — Дзен предложит сгенерировать из инициалов, потом заменишь).
   - **Категория:** «Производство и промышленность» → подкатегория «Металлургия».
   - **Тэги:** металл, металлопрокат, B2B, строительство, ГОСТ.
6. Жми «Создать канал». ✅ Готово — у тебя есть канал, URL вида `dzen.ru/harlansteel`.

### Публикация первого поста

1. На странице канала — кнопка «Создать публикацию» (правый верхний угол).
2. Выбери тип: **«Статья»** (не «Краткая публикация», не «Видео»).
3. Открой файл `content/distribution/ready-to-publish/01-dzen-kak-bezopasno-pokupat.md` — копируй заголовок в поле «Заголовок».
4. Скопируй полный body в редактор Дзена. Markdown поддерживается частично — проверь форматирование (H2/H3, **bold**, списки).
5. Загрузи hero image: возьми из `metallportal/public/images/blog/kak-bezopasno-pokupat-metall-optom-hero.jpg` (после Кати session).
6. Внизу — settings:
   - **Тэги:** металл, B2B, снабжение, ГОСТ, контрагент, проверка поставщика
   - **Категория:** Бизнес → Малый и средний бизнес
   - **Возрастное ограничение:** 0+
7. Жми **«Опубликовать»** (не «Сохранить как черновик»).
8. Скопируй URL опубликованной статьи → отправь Юле.

### Cadence Я.Дзен

- **Частота:** 2-3 поста в неделю (понедельник + среда + пятница оптимально).
- **Лучшее время:** 9:00-10:00 MSK (утро понедельника — пик активности на Дзене).
- **Длина:** 800-1500 знаков для прокладывающих статей (не лонгриды — Дзен алгоритм режет attention выше 2000).
- **Hashtag-стратегия:** 3-5 hashtags релевантных + 1-2 broader (например, #B2B #предпринимательство).

### Best practices Я.Дзен

- **Заголовок** — обязательно содержит цифру или вопрос («7 проверок» / «Как выбрать»). Это даёт +30-50% CTR в фиде.
- **Lead paragraph** — первые 150 знаков критичны (виден в preview без раскрытия). Hook + цифры + что узнает читатель.
- **Image at start** — обязательно, без image статья получает 50-70% меньше показов.
- **CTA в конце** — ссылка на полный гид на harlansteel.ru/blog/<slug>. Дзен не любит external links — упоминай как «Подробнее — в нашем гиде» без явного `harlansteel.ru` в начале (модерация хуже).

   - **Username:** `harlansteel` (доступность проверится автоматически; если занято — `harlansteel_official` / `harlan_metall`).

### Публикация первого поста

1. Открой канал в Telegram.
2. Открой файл `content/distribution/ready-to-publish/02-tg-anatomiya-buy-box.md`.

6. Жми «Отправить». ✅ Опубликовано.

8. Скопируй URL → отправь Юле.

### Cadence Telegram

- **Частота:** 5-7 постов в неделю (Telegram любит регулярность, мёртвый канал хуже).
- **Лучшее время:** 18:00-19:00 MSK (пик активности B2B-аудитории после рабочего дня).
- **Длина:** 300-500 знаков (короткие посты лучше работают в Telegram fiд).
- **Hashtag-стратегия:** 3-5 hashtags в конце поста (#металлопрокат #B2B #снабжение #харлансталь и темовые типа #арматура / #профнастил).

### Best practices Telegram

- **Emoji в начале** — обязательно (🏗 / 🔩 / ⚙️ / 📊). Без emoji пост теряется в feed'е.
- **3 ключевых факта bullet'ами** — формат «3 факта + ссылка на full guide». Это самый эффективный pattern для B2B Telegram.
- **Ссылка на сайт в конце** — Telegram автоматически генерирует preview, это бонусная видимость.
- **Длинные тексты — через formatting** (`<b>`, `<i>`, `<code>`) — Telegram поддерживает HTML / Markdown.

- **Опросы** — VK любит interactive контент. Раз в месяц можно делать опрос «Какая статья вам бы пригодилась» с 4-5 топиками.
- **Stories** — VK Stories показываются за пределами сообщества. Использовать для anonsing новых статей.
- **Wiki-страницы** — для evergreen контента (например, главная страница «Все наши гайды по металлу» с links на 16 articles).

## 4. VC.ru — 5 минут регистрация + 2 минуты публикация

### Регистрация автора

1. Открой https://vc.ru → правый верхний угол → «Войти» / «Зарегистрироваться».
2. Зарегистрируйся через Yandex / Google / Email (рекомендую Email с корпоративным `info@harlansteel.ru`).
3. После входа — настрой профиль:
   - **Имя:** `Харлансталь` (компанейский профиль) или личное имя Sergey'я с пометкой «CEO Харлансталь»
   - **Описание (bio):** `B2B-маркетплейс металлопроката. Делюсь опытом построения B2B-tech продукта в индустрии металла.`
   - **Аватар:** logo Харлансталь
   - **Ссылка на сайт:** https://harlansteel.ru
4. Опционально: создай **компанейский профиль** (бесплатно для базового) — отдельная страница `vc.ru/u/harlansteel`. Постится от имени компании, выглядит профессиональнее.

### Публикация первого поста

1. В шапке VC.ru — кнопка «+ Написать».
2. Выбери раздел: **«Бизнес»** или **«Технологии»** (Buy Box подходит под Tech, остальные — Бизнес).
3. Открой файл `content/distribution/ready-to-publish/04-vc-buy-box.md`.
4. Скопируй заголовок и body.
5. VC.ru использует свой WYSIWYG-редактор — markdown поддерживается частично, проверь H2/H3, bold, lists.
6. Загрузи cover image (обязательно — без неё статья получает -50% reach).
7. **Tags:** 3-5 релевантных (металл, B2B, маркетплейс, снабжение, technology).
8. **Рубрика:** «Бизнес» (подрубрика — Промышленность) или «Технологии» (Tech / SaaS).
9. Жми «Опубликовать». VC.ru имеет модерацию — статья может появиться через 30 минут — 2 часа после publish.
10. URL: `vc.ru/<category>/<id>-<slug>` → скопируй после публикации.

### Cadence VC.ru

- **Частота:** 1-2 поста в неделю (VC.ru — качественный, не количество).
- **Лучшее время:** 10:00-11:00 MSK или 16:00-17:00 (рабочее время B2B).
- **Длина:** 1500-3000 знаков (VC.ru аудитория готова к лонгридам).
- **Hashtag-стратегия:** 3-5 теги, релевантные.

### Best practices VC.ru

- **Заголовок** — провокационный или с цифрой («Как мы делаем X», «5 ошибок при Y»).
- **Lead paragraph** — обязательно «крючок» (проблема + что я расскажу).
- **B2B-tone** — экспертный, с конкретными цифрами. VC.ru аудитория квалифицированная, не любит маркетинговую вату.
- **CTA** — без агрессивного «купите» (Vc.ru модерация банит за рекламу). Лучше «больше деталей в нашем гиде» с ссылкой на блог.
- **Комментарии** — обязательно отвечать на первые 5-10 комментов в течение 2 часов после публикации (boost в feed).

## 6. Pikabu — 5 минут регистрация + публикации Week 2+

### Регистрация

1. Открой https://pikabu.ru → «Регистрация».
2. Через Email или Yandex.
3. Заполни никнейм (например, `harlansteel`).
4. ✅ Готово.

### Стратегия публикаций

Pikabu — массовая аудитория, не B2B. Подходит для **кейсов «как обманывают на металле»**, **fail-stories**, **советов в формате «лайфхак»**. Не подходит для строго технического контента (паспорта качества, ГОСТ vs ТУ — будет downvote).

**Format-friendly для Pikabu:**
- Истории о работе со снабженцами (без раскрытия конфиденциальной информации)
- Сравнения «дорого vs дёшево» в металле
- Чек-листы «как не попасть на мошенника при покупке металла» (адаптация art #6)

### Cadence Pikabu

- **Частота:** 1-2 поста в месяц.
- **Лучшее время:** 19:00-21:00 MSK (вечерний пик).
- **Длина:** 1500-3000 знаков.

### Best practices Pikabu

- **Заголовок** — кликбейтный, эмоциональный («Как я потерял 300 тысяч на металле — и что понял»).
- **Истории от первого лица** работают лучше chистого экспертного гайда.
- **Image gallery** — Pikabu любит несколько изображений в посте.
- **Tags:** 3-5 (металл / бизнес / истории).
- **Без явной рекламы** — Pikabu модерация банит за продвижение. Ссылку на сайт — только в bio.

## Troubleshooting (если что-то не работает)

**Я.Дзен не публикует статью / модерация отклонила:**
- Проверь что нет голых external links в первых 100 знаках текста (Дзен flag'ает как реклама).
- Замени `harlansteel.ru` в начале на «нашем гиде» (без явной ссылки).
- Если повторно отклонена — пинг в Дзен support через личный кабинет.

**Telegram не показывает preview ссылки:**
- Подожди 30 секунд после отправки — Telegram парсит OG meta с лагом.
- Если preview не появляется через 5 минут — ссылка на /blog/<slug> возможно ещё не имеет правильных meta tags (пинг Никите про blog reader).

**VK блокирует пост за «нарушение правил»:**
- Проверь hashtags — не более 10. VK банит за hashtag spam.
- Убрать ссылку из первой строки поста (перенести в середину или конец).

**VC.ru держит на модерации больше 24 часов:**
- Это normal для первой публикации нового автора.
- Если 48+ часов — пинг VC support через личный кабинет.

**Habr не отвечает на заявку:**
- Срок 5-14 дней.
- Если 14+ дней — повторная заявка с уточнениями.

## FAQ для Sergey'я

**В: Можно ли постить одну и ту же статью на все 6 платформ?**
О: Технически да, но это бесполезно — каждая платформа имеет свою аудиторию и формат. Используй готовые adaptations в `content/social/<slug>/` — Юля адаптировала под каждый канал отдельно.

**В: Что если Я.Дзен не дал монетизацию?**
О: Это нормально для нового канала. Монетизация открывается после 100+ подписчиков и 7 публикаций. До этого — фокус на growth, не на доходе.

**В: Можно ли пропустить какой-то канал?**
О: Да, но не рекомендую. Каждый канал = отдельная аудитория. Telegram + VK + VC.ru = минимальный набор для B2B (если только один — Telegram). Я.Дзен — для широкого reach (mixed audience).

**В: Что если у меня нет hero images?**
О: Постить можно без image, но reach будет в 2-3 раза ниже. Лучше попросить Катю срочно сделать placeholder hero (gradient + текст с заголовком) — она это делает за час. Или использовать stock photos.

**В: Сколько времени на ВСЕ публикации в неделю?**
О: ~30-60 минут в неделю при использовании готовых файлов из `ready-to-publish/`. Главное — следовать cadence chart, не публиковать всё сразу или с большими паузами.
